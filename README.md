How To Use
==========
Exit: esc
Zoom in: i
Zoom out: o
Movement: Arrows or WASD
Rotatation: x-axis 1 2 / y-axis 3 4 / z-axis 5 6
Z_scale: - +
Change Projection: p
Change Color: c v
Start/Stop: space
Disko: k
Reset: r or 0
Hide/Show Menu: Tab, m or h
